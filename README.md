# vityarthi_project_25BCY10096  

PROJECT TITLE : STUDENT PERFOARMANCE ANALYZER  
  
OVERVIEW  
This project provides a Python Script to load student marks from a CSV file, analyse the performance through statitical measures, and visulize the results using a bar chart. It hepls educators or analysts quickly understand student performance distribution.  
  
FEATURES  
Load student names and marks from a CSV file.  
Calculate key statistics: average median highest lowest marks and standard deviation  
Generate a bar chart visualizign marks for each student  
Handles errors such as missing input file gracefully  

TOOLS USED  
Python 3  
Modules: csv statistic matplotlib   

INSTRUCTIONS TO RUN  
Ensure Python 3 is installed on your system.    
Install matplotlib  
Place your csv file in the project directtory  
Run the Python script:  
python VITsneha.ipynb  

TESTING INSTRUCTION  
Prepare a CSV file named Book1.csv or change the filename in the script accordingly.  
Ensure the CSV file is formatted correctly.  
Run the script and verify the terminal output includes statistical measures.  
Check that the bar chart window appears showing the student marks.
